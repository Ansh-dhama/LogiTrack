# LogiTrack - Logistics Management System
A Spring Boot backend for tracking shipments and managing logistics.

## Tech Stack
- Java 17+
- Spring Boot 3
- Spring Security & JWT
- Spring Data JPA
- SQL (MySQL/PostgreSQL)
- Swagger UI (Documentation)

## How to Run
1. Clone the repo.
2. Update `application.properties` with your DB credentials.
3. Run `./mvnw spring-boot:run`.
4. Access Swagger at `http://localhost:8080/swagger-ui.html`.
